# 🤖 AMES: Automated Machine Learning Expert System

> An end-to-end AutoML system that automatically selects, tunes, and evaluates the best machine learning algorithm for any given classification or regression dataset — **published at COM-IT-CON 2022**.

---

## 📄 Publication

> **P. S. Patil, K. Kappuram, R. Rumao, P. Bari** — *Development of AMES: Automated ML Expert System*
> Published at **COM-IT-CON 2022** · [View Paper](https://ieeexplore.ieee.org/document/9850909)

---

## 🎯 Problem

Building a machine learning model for a new dataset typically requires significant manual effort — choosing the right algorithm, engineering features, tuning hyperparameters, and evaluating performance. AMES automates this entire workflow, enabling faster and more consistent model development across classification and regression tasks.

---

## ⚙️ What AMES Automates

| Task | Description |
|------|-------------|
| **Data Pre-processing** | Handles missing values, encoding, and scaling automatically |
| **Feature Engineering** | Selects and transforms relevant features for the task |
| **Model Selection** | Benchmarks multiple algorithms and picks the most accurate |
| **Hyperparameter Optimisation** | Tunes model parameters for optimal performance |
| **Visualisation** | Generates performance plots and model comparison charts |

---

## 🔍 Supported Tasks

- **Classification** — binary and multi-class
- **Regression** — continuous target prediction

## 🚀 How to Run

1. Clone the repository
```bash
git clone https://github.com/keertanakappuram/Automated-Machine-Learning-Expert-System.git
cd Automated-Machine-Learning-Expert-System
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run AMES on your dataset
```bash
cd "Automated Machine Learning Expert System"
jupyter notebook
```
